# Get started

Golang [Downloads](https://go.dev/dl/)

```
go version
```

Go mod [libery](https://pkg.go.dev/std)

```
go mod init name
```

---

## Run and build

run

```
go run .\index.go
```

execution file

```
go build ./file
```

---

## Automatically restarting

- window

  node mon

  ```
  npm i -g nodemon
  ```

  ```
  nodemon --exec go run .\index.go
  ```

- mac

  ```
  npx go-mon name.go
  ```

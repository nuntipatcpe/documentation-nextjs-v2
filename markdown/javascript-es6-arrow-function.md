# Arrow Function

- Original Function
  <code language="javascript">
  function originalName(p,p1){
  return p+p1;
  }</code>

- Arrow Function **Recommend**
  <code language="javascript">
  originalName = (p,p1)=>{
  return p+p1;
  }</code>

  or
  <code language="javascript">
  originalName = (p,p1) => p+p1
  </code>

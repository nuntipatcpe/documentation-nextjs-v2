# Linux

## Folder

- Make Directory

```
mkdir nameFolder

```

- Remove Directory

```
- rmdir nameFolder

```

- Remove force Directory

```
rm -rf nameFolder
```

## Show File

```
ls

```

## Show Path File

```
pwd

```

```
cd nameFolder

```

cd ..

# Media query

- header

  ```
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  ```

- Media Query

  max-width

  ```
  @media screen and (max-width: 400px) {
    body {
      background-color: lightgreen;
    }
  }
  ```

  min-width

  ```
  @media screen and (min-width: 400px) {
    body {
      background-color: lightgreen;
    }
  }
  ```

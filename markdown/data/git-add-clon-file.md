# ADD File

- Install git and add to git staging area

  ```
  git init
  git add .
  ```

- check status

  ```
  git status
  ```

- Commit and change name main branch
  ```
  git commit -m "first commit"
  git branch -M main
  ```
  ```
  git remote add origin https:\/\/github.com/..
  ```
  ```
  git push -u origin main
  ```

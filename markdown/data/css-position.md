# Position

- Static **default**

  ```
  position: static
  ```

- Relative **box parent**

  ```
  position: relative
  ```

- Absolute **Position by box parent (relative)**top , bottom , left , rigth

  ```
  position: absolute
  ```

- Fixed **set width & heigth**

  ```
  position: fixed
  ```

- Sticky **set Position** top , bottom , left , rigth

  ```
  position: sticky
  ```

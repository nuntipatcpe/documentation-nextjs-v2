# Branch

- Check branch

  ```
  git branch
  ```

- **Change** name

  ```
  git branch -M main
  ```

- **Create**

  ```
  git branch -c new_branch
  ```

- **Delete**

  ```
  git branch -d newBranch
  ```

- **Force delete**

  ```
  git branch -D newBranch
  ```

- **Switch**

  ```
  git switch  <non_existing_branch>
  ```

- **Switch** and create new branch "-c"

  ```
  git switch -c <non_existing_branch>
  ```

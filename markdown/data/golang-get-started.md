# Get started

- Golang [Downloads](https://go.dev/dl/)

  ```
  go version
  ```

- Go mod [libery](https://pkg.go.dev/std)

  ```
  go mod init name
  ```

- Run

  ```
  go run .\index.go
  ```

- Build **execution file**

  ```
  go build ./file
  ```

# React Datepicker

- Install [documentation](https://reactdatepicker.com/)

  ```
  pnpm install react-datepicker --save
  pnpm i @types/react-datepicker
  ```

- Format

  ```
  date.toLocaleDateString("en-US", {
            // weekday: "short",
            year: "numeric",
            month: "short",
            day: "numeric",
          })
  ```

# Reduce

- Paramiter cur = currentValue
- Paramiter pre = previousValue
- Paramiter init value = initialValue

  Syntax

  ```
  const summation  = data.reduce((pre,cur)=>{
      const total = e+value;
      return total;
  },init)
  ```

  Example

  ```
  const data = [10,20,30]
  const summation  = data.reduce((pre,cur)=>{
      const total = cur+pre;
      return total;
  },9)
  console.log(summation)
  ```

  output

  ```
  69
  ```

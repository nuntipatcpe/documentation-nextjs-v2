# Install

- package.json

  ```
  npm init -y
  ```

- nodemon

  ```
  npm i nodemon
  ```

- run nodemon server.js

  ```
    "scripts": {
      "test": "echo \"Error: no test specified\" && exit 1",
      "start": "nodemon server.js"
    }
  ```

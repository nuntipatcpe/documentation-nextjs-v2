# Security Operation Center(SOC)

- is the facility where the information security team constantly monitors and analyzes the security of an organization.

- detect, analyze and respond to cyber security

# Types Of SOC Models

- In-house SOC
- Virtual SOC
- Co-Managed SOC
- Command SOC

# SOC Positions

- SOC Analyst
- Incident Responder
- Threat Hunter
- Security Engineer
- SOC Manager

# Type

- example

  ```
  export type Test = {
    array: Array<{
      name: string;
      number: number;
    }>;
  }
  ```

- Array

  ```
  array: Array<{}>;
  ```

  ```
  array: Array<string>;
  ```

  ```
  array: [];
  ```

- Function

  Return Void

  ```
  func:()=> void
  ```

  Return String

  ```
  func:()=> string
  ```

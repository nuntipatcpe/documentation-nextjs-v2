# Colors

- name

  ```
  color : red;
  ```

- rgb(red,green,blue)

  ```
  color : rgb(0,0,255);
  ```

- Hexadecimal

  ```
  color : #000;
  ```

- rgba ( red , green , blue , 0.2 )

  ```
  color : rgb(0,0,255,0.5);
  ```

- HSL :(hue,saturation,lightness)

  ```
  background-color: hsl(27, 100%, 50%);
  ```

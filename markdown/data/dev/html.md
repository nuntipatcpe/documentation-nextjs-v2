# Table

- Layout

  - table <**table**>
  - table header <**thead**>
  - table body <**tbody**>
  - table footer<**tfoot**>

- data

  - table data head <**th**>
  - table row <**tr**>
  - table data <**td**>

- example

  ```
  <table>
    <thead>
        <tr>
            <th>Header</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>data1</td>
            <td>data2</td>
            <td>data3</td>
        </tr>
    </tbody>
    <tfoot></tfoot>
  </table>
  ```
